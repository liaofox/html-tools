# HTML小工具集合

一个收集了各种实用HTML小工具的网站，所有工具都在浏览器本地运行，无需服务器支持，保护用户隐私。

## 🌟 特点
- 🚀 完全在浏览器中运行
- 🔒 不收集任何用户数据
- 📱 响应式设计，移动端友好
- 🎨 简洁美观的界面
- 🆓 完全开源免费

## 🛠️ 包含的工具
1. **科学计算器** - 支持基本运算和科学计算
2. **颜色选择器** - 可视化颜色选择，支持多种格式
3. **单位换算器** - 长度、重量、温度等单位转换
4. **文本格式化** - 文本处理工具
